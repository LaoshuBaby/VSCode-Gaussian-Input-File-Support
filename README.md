# Gaussian Input File Support

A Visual Studio Code extension for Gaussian input file (aka. `*.gjf` file) with syntax highlight support.

## Features

- Basic syntax highlight
    - Link 0 commands
    - Keyword options
    - Methods
    - Basises
    - Atom symbols
    - Numbers
